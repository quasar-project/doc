quasar::IAbstractRadar class
#include <quasar/radar/abstract_radar.h>

Абстрактный интерфейс радара.

Пример использования в Python:

import quasar

# ... аргументы ...

# инициализация радара с использованием конфигурационного файла и файла голограммы
radar = quasar.RadarSTT(args.paths['config'], args.paths['data'])

# поиск синхропаузы
pos, width = radar.sync_pos(args.time_shift, trash_offset)

# чтение голограммы для работы в телескопическом режиме
image = radar.read_image(radar.period_count(args.synthesis_time), args.time_shift, pos,
args.fic, False, True)

# поблочное чтение голограммы для полосового режима
block_image = radar.block_init(nb, args.fic, False)for k in range(nstrip):
block_chunk = radar.block_read(nb, args.fic)
bp.set_image(block_chunk)

auto quasar::IAbstractRadar::read_image(usize period_count, f32 time_shift, usize sync_pos, i32 freq_interpolation_coeff, bool round, bool remove_transition) const pure virtual noexcept(…)

Производит считывание данных голограммы из заданного файла в новый объект изображения согласно параметрам текущего радара.

Parameters
period_count Количество периодов для считывания.
time_shift Смещение стартовой позиции для чтения (в секундах).
sync_pos Смещение до синхропаузы от начала файла (в отсчётах).
freq_interpolation_coeff Коэффициент частотной интерполяции.
round Режим округления количества отсчетов (округлять до следующей степени двойки или нет).
remove_transition Удаление переходного процесса.
Returns Возвращает новый объект изображения в случае успеха.
Exceptions
std::logic_error Если текущий экземпляр не был инициализирован файлом голограммы.
std::invalid_argument Если параметры заданы некорректно (например, отрицательное смещение).
std::runtime_error Если не удалось считать голограмму.

Note

Если задан параметр remove_transition, то будет выполнена проверка на наличие синхропаузы размером sync_samples_count в начале позиции для считывания.

Invariant

Доступно в Python в виде функции.

auto quasar::IAbstractRadar::find_sync_pos(CImage const& image) const pure virtual noexcept(…)

Производит поиск паттерна синхропаузы в существующем изображении

Parameters
image Ссылка на действительное изображение.
Returns Кортеж с индексами начала и конца синхропаузы в случае успеха, сообщение об ошибке в случае неудачи.
Exceptions
std::invalid_argument Если переданное изображение имеет неподдерживаемый тип данных.
std::runtime_error Если не удалось произвести поиск синхропаузы.

Первое значение в результирующем кортеже - это количество отсчётов от начала буфера изображения до начала синхропаузы. Второе значение - это количество отсчётов от начала буфера изображения до конца синхропаузы. Если синхропауза не обнаружена, то функция выбросит исключение.

Note

Если переданное изображение имеет тип данных в буфере, отличный от float32, то функция выбросит исключение.

Invariant

Доступно в Python в виде функции.

auto quasar::IAbstractRadar::sync_pos(f32 time_shift, usize offset) const pure virtual noexcept(…)

Загружает часть изображения в буфер и производит поиск синхропаузы.

Parameters
time_shift Смещение стартовой позиции для чтения (в секундах).
offset Смещение до синхропаузы от начала файла (в отсчётах).
Returns Кортеж с индексами начала и конца синхропаузы в случае успеха, сообщение об ошибке в случае неудачи.
Exceptions
std::invalid_argument Если параметры заданы некорректно (например, отрицательное смещение).
std::runtime_error Если не удалось считать голограмму.
std::logic_error Если текущий экземпляр не был инициализирован файлом голограммы.

Эквивалентно следующему коду в Python:

image = radar.read_image(2, time_shift, offset, -1, False, False)
pos, width = radar.find_sync_pos(image)
pos += offset
return pos, width
