enum class quasar::DSP
#include <include/quasar/api/dsp.h>

Перечисление возможных типов вычислителя.

Enumerators
FFTW

Вычислитель на основе библиотеки FFTW3.

Выполняется на ЦП.
CUDA

Вычислитель на основе NVidia CUDA API.

Выполняется на графическом процессоре. Поддерживаемые архитектуры CUDA:

  • CUDA 7.2 (NVidia Jetson Xavier)
  • CUDA 8.6 (NVidia RTX 30XX series)

  • CUDA 8.7 (NVidia Jetson Orin Nano)

    Remark

    Доступен с feature-flag QUASAR_CUDA - см. README.md для деталей по сборке проекта.
