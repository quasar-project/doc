Копирует коллекцию одного типа в новый вектор другого типа.

Template parameters
To Тип элемента результирующего вектора. Должен быть арифметическим типом (целое или вещественное число).
InputIt Тип итератора исходной коллекции. Должен хранить арифметический тип (целое или вещественное число).
Parameters
begin Начало исходной коллекции.
end Конец исходной коллекции.
Returns Новый вектор размером std::distance(begin, end) c типом элемента To.

template<typename InputIt>
auto quasar::ranges::mean_stddev(InputIt begin, InputIt end) -> std::pair< typename std::iterator_traits< InputIt >::value_type, typename std::iterator_traits< InputIt >::value_type >

Вычисляет среднее значение и СКО для диапазона.

Template parameters
InputIt Тип итератора исходной коллекции. Должен хранить арифметический тип (целое или вещественное число).
Parameters
begin Начало исходной коллекции.
end Конец исходной коллекции.
Returns Среднее значение и СКО для диапазона в виде пары элементов.

Пример использования:

auto vec = std::vector<int>{1, 2, 3, 4, 5};
auto const [mean, stddev] = ranges::mean_stddev(vec.cbegin(), vec.cend());
std::cout << mean << " " << stddev << std::endl;
