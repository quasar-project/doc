result quasar::Image::cut(Dimension mode, usize start, usize end)

bool quasar::Image::is_subnormal() const

Возвращает true, если в буфере данных есть NaN или Inf-значения.

Returns true, если в буфере данных есть NaN или Inf-значения.

Проверяет наличие NaN/Inf-значений в буфере данных.

Алгоритмическая сложность вычисления: O(n), где n - размер буфера.

Note

Вызов данного метода затратен по времени, поэтому его следует использовать осторожно.

Image& quasar::Image::operator=(Image&& other) protected noexcept

Оператор перемещения.

Выполняет перемещение изображения в новый экземляр, инвалидируя предыдущий. Буфер при этом не копируется, а перемещается, т.е. затраты по скорости минимальны.

Remark

Это - операция перемещения, а не копирования. Она инвалидирует предыдущий экземпляр, поэтому обеспечивает минимальные затраты по скорости.
