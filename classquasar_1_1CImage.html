Создает пустой экземляр изображения выбранного типа.

Parameters
type Тип DSP. Влияет на место аллокации памяти: в случае DSP::FFTW буфер изображения создается в выделенной оперативной памяти, в случае DSP::CUDA - в видеопамяти.

quasar::CImage::CImage(DSP type, u32 width, u32 height, ImageUnderlyingType underlying_type) explicit

Создает экземляр изображения выбранного типа c преаллоцированными размерами width x height и выбранным типом отсчета.

Parameters
type Тип DSP. Влияет на место аллокации памяти: в случае DSP::FFTW буфер изображения создается в выделенной оперативной памяти, в случае DSP::CUDA - в видеопамяти.
width Ширина изображения в количестве отсчетов.
height Высота изображения в количестве отсчетов.
underlying_type Тип данных, которыми представлены отсчеты в буфере. В случае ImageUnderlyingType::Undefined будет аллоцированно пустое изображение.

See also

Конструктор копирования.

Parameters
other Копируемое изображение.

Выполняет глубокое копирование изображения (копирование буфера) в новый экземляр.

Warning

Копирование изображения - затратная операция. Используйте ее только при необходимости.

Version

quasar::CImage::CImage(CImage&& other) noexcept

Конструктор перемещения.

Parameters
other Перемещаемое изображение.

Выполняет перемещение изображения в новый экземляр, инвалидируя предыдущий. Буфер при этом не копируется, а перемещается, т.е. затраты по скорости минимальны.
